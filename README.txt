
3D-Print Store — Demo (Shopify-like)
===================================

Inhalt:
- index.html : Shop-Frontend
- admin.html : Admin-Panel (Demo, client-side auth)
- css/styles.css, js/shop.js : Assets

Admin Passwort (Demo): chat5test
Wichtig: Dies ist eine lokale Demo. Alle Bestellungen und der Warenkorb werden **nur** in deinem Browser (localStorage) gespeichert — es gibt kein Backend und keine echte Zahlung.

Anpassungen:
- Produkte sind in js/shop.js in der PRODUCTS-Variable definiert.
- Um echte Checkout-Integration zu bauen (Stripe/etc.), brauchst du ein Server-Backend und API-Schlüssel.
